经过三周多的实践，阅读完了《Unity Shader 入门精要》，做了相应的读书笔记作为记录，完成了书中的实践，最后录制了个视频demo，并把项目文件上传至 Github 仓库。我看是完成了。那么接下来做什么呢？

现状

  • 了解了渲染用到的数学，渲染管线，光照模型等基础知识
  • 上述在 Unity 中的实现
  • 还没用过 Unreal Engine
  • 还不会开发优化流程的工具
  • 还没完成对 OpenGL、DirectX 的学习

下一步的方向

  • 百人计划视频，知识点区分明显，可以过一遍，同时做笔记
  • 在 Unreal Engine 中实践
  • 光线跟踪、曲线曲面、计算机动画（清华大学-计算机图形学基础）
